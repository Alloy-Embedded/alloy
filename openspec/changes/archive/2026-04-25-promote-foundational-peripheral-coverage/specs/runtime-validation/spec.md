## ADDED Requirements

### Requirement: Peripheral Class Promotions Shall Be Validation-Backed

Promotions of peripheral support tiers SHALL be backed by the matching validation ladder.

#### Scenario: Promoting a representative peripheral class

- **WHEN** a peripheral class moves from representative to foundational
- **THEN** the repo already has the required smoke, host/emulation, example, and hardware evidence
- **AND** the support matrix changes in the same change
