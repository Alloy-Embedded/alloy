## ADDED Requirements

### Requirement: Supported Recovery And Debug Flows Shall Be Public

Release documentation SHALL publish the supported recovery and debug flows for each foundational
board that is part of the supported product story.

#### Scenario: User needs to recover a foundational board

- **WHEN** a supported foundational board requires recovery after a bad flash
- **THEN** the documented tooling flow is public and maintained
- **AND** it is not left to private operator knowledge
